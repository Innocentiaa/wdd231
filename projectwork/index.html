Site Purpose

This website serves as a fun and informative space to discover top Nollywood movies, learn more about Nigerian film culture, and explore curated recommendations. It includes a homepage introduction, top picks powered by the OMDB API, and a movie details section with watchlist features and cultural commentary.

Scenarios

  • Which Nollywood movie should I watch this weekend?
  • How can I find more details about a Nigerian film I recently heard about?

Color Schema

  • Deep Navy Green (#00332b): Used for headings, main text, and primary elements for excellent contrast.
  • Deep Gold (#bfa000): Used for accents, borders, and highlight elements to add a rich, elegant feel.
  • Soft Cream (#fff8f0): Used as the background color to create a warm and welcoming look.

Typography

  • Montserrat (Sans-serif): Used for major headings and titles.
  • Segoe UI: Used for body text and paragraphs.
